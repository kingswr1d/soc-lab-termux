# Contributing

Small, focused contributions are welcome.

## Before you start
- Open an issue describing the change.
- Keep scripts dependency-light; Python stdlib + scapy/requests/python-nmap/yara-python only.

## What counts as a good PR here
- A new detection script with sample data
- A lab writeup in a markdown file
- A bugfix with before/after output in the description

## Style
- 2-space indentation
- Docstring on every function
- No secrets or real credentials
